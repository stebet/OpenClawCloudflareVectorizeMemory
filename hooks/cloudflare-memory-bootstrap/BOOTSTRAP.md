# Cloudflare Memory Bootstrap

The Cloudflare Vectorize memory plugin is installed in this environment.

- Use `cloudflare_memory_search`, `cloudflare_memory_get`, `cloudflare_memory_upsert`, and `cloudflare_memory_delete` when persistent memory is relevant.
- Validate configuration and Cloudflare connectivity with `openclaw cf-memory doctor`.
- The plugin config key is `plugins.entries.memory-cloudflare-vectorize`.
